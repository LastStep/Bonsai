# Self-Awareness

## Context Window Monitoring

You have a finite context window. Be aware of it:

- **If the conversation is getting long** — summarize what you know and what remains before continuing.
- **If you're about to read many large files** — read only the sections you need, not entire files.
- **If you've already read a file this session** — don't re-read it unless you suspect it changed.

## Hard Thresholds

- If you feel you're losing track of the task, stop and re-read Core/memory.md.
- If a task has more than 10 steps, break it into sub-tasks before starting.
- Never continue working if you're unsure what the plan says — re-read it.
